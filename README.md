# 🏆 GoldBullionTrade.com - Real-Time Precious Metals Dashboard

An advanced, high-performance financial dashboard for tracking global gold spot prices and calculating bullion values in real-time. Designed for premium investors and precious metal traders.

## 🚀 Key Features
- **Live Market Data:** Integrated with **GoldAPI.io** for real-time XAU/USD spot prices.
- **Smart Calculator:** Precision tool for calculating 24K, 22K, 21K, and 18K gold values based on live weight.
- **Modern UI/UX:** Built with **Tailwind CSS** featuring a luxury Dark Mode and glassmorphism effects.
- **Compliance Ready:** Includes built-in modals for Privacy Policy, Terms of Service, and Contact Desk.
- **Optimized for Ads:** Clean structure designed for Google AdSense approval.

## 🛠️ Tech Stack
- **Frontend:** HTML5, Tailwind CSS (via CDN)
- **Engine:** Vanilla JavaScript (ES6+)
- **API:** [GoldAPI.io](https://www.goldapi.io/)
- **Hosting:** Vercel / GitHub Pages
- **Communication:** Formspree.io Integration

## 📈 Financial Data Coverage
The dashboard provides automated price updates for:
* **Gold 24K** (99.9% Purity)
* **Gold 22K** (Standard Bullion)
* **Gold 21K** (Investment Grade Jewelry)
